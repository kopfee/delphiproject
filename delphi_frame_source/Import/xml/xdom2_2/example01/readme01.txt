FILE DESCRIPTION

This example source code shows how to 
load and display the source of XML files
using the XDOM Delphi unit. In order to
have it run, XDOM version 2.2.1 or above 
must be installed on your Delphi system. 
You can obtain the latest XDOM version
at "http://www.philo.de/xml/" free of 
charge.
